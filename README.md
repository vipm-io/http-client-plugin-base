# http-client-baseclass

<img width="400" alt="image" src="https://github.com/user-attachments/assets/0950f5ae-8660-4de0-9669-6c7490725359" />


A base class interface identical to the built-in HTTP Client, allowing easy substitution of other implementations. Defaults to LabVIEW's built-in HTTP Client implementation.

There are plugins for [NI Advanced HTTP Client](https://www.vipm.io/package/vipm_io_http_client_plugin_ni_advanced/) as well as (coming soon) JKI's [request-rs-labview](https://github.com/JKISoftware/reqwest-rs-labview/) library
